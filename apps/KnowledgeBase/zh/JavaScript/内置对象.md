

# 内置对象

是所有对象的基类，Object的键只能是字符串或者symbol类型，如果写成其他类型会自动转化为字符串；

访问对象属性可以用 点（`.`） 或 方括号（`[]`） 两种方式，区别：

点（`.`）访问属性：

- 属性名必须是合法的标识符，不能是数字开头、空格、特殊字符等

- 不能动态访问：访问时，JS会直接找key字符串对应的属性名

  ```
  let key = "name";
  console.log(obj.key); //  undefined
  ```

方括号（`[]`）访问属性：

- 属性名可以是任意字符串或表达式

- 可以动态访问：`[]` 内可以放变量、表达式、字符串等，JS 会先计算 `key` 的值，再去对象里找这个属性

  ```
  let key = "name";
  console.log(obj[key]); // ✅ "Alice"
  ```

  

#### delete

一个操作符，可以根据对象的键删除属性（实际上转化为undefined）

```
// 删除对象属性
const obj = { a: 1, b: 2 };
delete obj.a; 
console.log(obj.a); 

// 删除数组元素
const arr = [10, 20, 30];
delete arr[1];
console.log(arr); // [10, undefined, 30]（元素被删除，但数组长度不变）
```

#### hasOwnProperty()

根据属性名判断属性是否是对象自身上的，而不是原型链上的，返回bool值；只能检查第一层属性；

```
let obj = {
    name:'aaa',
    age:18,
    hobby:['篮球','乒乓球'],
    family:{
        fahter:'bbb',
        mother:'ccc',
    }
}
console.log(obj.hasOwnProperty('name'));//true
console.log(obj.hasOwnProperty('father'));//false
```

#### Object.getOwnPropertySymbols()

能获得对象中Symbol键数组；

```
const s1 = Symbol("id");
const s2 = Symbol("type");

let obj = {
    [s1]: '1',
    [s2]: 'bbb',
    name:'aaa'
}

let symbolArr =Object.getOwnPropertySymbols(obj);//输出[Symbol(id), Symbol(type)]
console.log(symbolArr[0]);//输出Symbol(id)
console.log(obj[s1]);//输出1
```

#### Object.keys()

获得对象中所有的属性名字符串的数组；

```
let obj ={name:'aaa',age:18}
let arr = Object.keys(obj);
console.log(arr);//arr为"name" "age"
```

#### Object.values()

获得对象中所有属性的值，返回一个数组；

```
let obj ={name:'aaa',age:18}
let arr = Object.values(obj);
```

#### Object.entries()

获得对象中属性键和值的二维数组

```
let obj = {
    a: 1,
    b: 2,
    c: 3
}
console.log(Object.entries(obj));//输出[ [ 'a', 1 ], [ 'b', 2 ], [ 'c', 3 ] ]
```

#### Object.assign()

将一个对象属性浅拷贝赋值给另一个对象，相同属性则会覆盖；

```
let obj1 ={name:'aaa',age:18}
let obj2 ={}
Object.assign(obj2,obj1);//将obj1的name，age加给obj2；
```

#### Object.defineProperty()

为对象定义新属性，或修改现有属性，并能设置属性的各种特性（如是否可修改、是否可枚举等）；第一个参数为操作的对象，第二个参数为操作的属性名，第三个参数为该属性的属性描述符配置（包括数据描述符和访问器描述符）；

**数据描述符：**

writable：是否允许修改属性的值，默认false；

enumerable：是否允许被枚举（如 `for...in`、`Object.keys()`），默认false

configurable：是否允许删除属性或修改描述符，默认false

```
let obj = {
    a: 1,
    b: 2,
    c: 3
}
Object.defineProperty(obj, 'd', {
    value: 4,
    enumerable: false,
    configurable: true,
    writable: true,
})
```

**访问器描述符**

get：当访问对象的某个属性时，自动触发的函数，用于返回属性值；

set：当修改对象的某个属性时，自动触发的函数，用于处理属性值的更新；

Object.defineProperty中不能同时指定 value/ writable 和 get /set；

```
let obj = {
    a: 1,
    b: 2,
    c: 3
}
Object.defineProperty(obj, 'd', {
    enumerable: true,
    configurable: true,
    get: function () {
        return 4
    },
    set: function (value) {
        console.log(value)
        return value;
    }
})
console.log(obj)//输出{ a: 1, b: 2, c: 3, d: [Getter/Setter] }
```

#### Object.getOwnPropertyDescriptor()

获得某个对象的某个属性的属性描述符

```
let obj = {
    a: 1,
    b: 2,
    c: 3
}
console.log(Object.getOwnPropertyDescriptor(obj, 'a'))
//输出{ value: 1, writable: true, enumerable: true, configurable: true }
```

#### Object.freeze()

使对象完全不可变（包括添加删除修改属性），可以使用Object.isFrozen来判断是否被freeze；

```
let obj = {
    a: 1,
    b: 2,
    c: 3
}
Object.freeze(obj)
console.log(Object.isFrozen(obj))
```

#### Object.seal()

限制对象的结构修改（如删除新增），但允许修改现有属性的值；可以用Object.isSealed判断是否被seal；

```
let obj = {
    a: 1,
    b: 2
}
Object.seal(obj)
console.log(Object.isSealed(obj))
```



# Number数字对象：

#### toFixed()

将数字格式化为指定位数的**小数字符串**，默认参数为0，结果会四舍五入

```
let float = 3.1415926;
let result = float.toFixed(2);//保留两位小数，结果为3，14
```

#### toString()

将数字转化为指定进制的**字符串**，默认十进制，进制范围为2到36；这原本是Object.prototyped上的方法，但是Number对其进行了重写

```
let a = 10;
console.log( a.toString(2));//二进制，结果为1010
```

#### Number.isInteger()

静态方法，判断传入值是否为整数，如果该值不是数字或是NaN，Infinity同样返回false；

```
let a = 10;
console.log(Number.isInteger(a));//true
```

# Array数组对象：

可以通过new Array来声明数组，参数可以为数组长度，数组元素

```
let arr = new Array(5);
let arr = new Array(5).fill(0);//长度为5，并且0填充

let arr = new Array(1, 2, 3, 4);// [1, 2, 3, 4]

#### 尾增：`push()`

返回新的length值；

```
let arr = [];
let a ={
    name:'aaa',
    age: 13
}
let b  ={
    name:'bbbb'
}
//追加复杂数据类型
arr.push(a);
arr.push(b);
arr.push({

#### 前增：`unshift()`

返回新的length值；

```
let arr =[1,2,3,4]
arr.unshift(-1,0);//arr变为[-1,0,1,2,3,4]
```

#### 尾删：`pop()`

返回删除元素的值；

```
let arr =[1,2,3,4,5];
arr.pop();//arr变为1,2,3,4
```

#### 前删：`shift()`

返回删除元素的值；

```
let arr =[1,2,3,4,5];
arr.shift();//arr变为2,3,4,5
```

#### 粘接：`splice()`

`array.splice(start, deleteCount, item1, item2, ...)`

- 第一个参数：操作的起始位置索引；

- 第二个参数：从起始位置开始要删除元素的个数；

- 往后的参数：从起始位置前开始要插入的元素；

直接操作数组，忽略返回值；

```
let arr =[1,2,3,4,5];

//删除，返回删除元素的数组
arr.splice(0,3);//arr变为[4,5]

//插入,第二个参数为0，表示只在index为2的地方开始插入6，7，8，而不删除元素
arr.splice(2,0,6,7,8);//arr变为1，2，6，7，8，3，4，5

//替换，第二参数不为0，表示从index为2开始删除2个元素再插入6,7;
arr.splice(2,2,6,7);//arr变为[1,2,6,7,5]
```

#### 拼接：`concat()`

不能直接使用 `+` 这样会将两个数组转化为字符串再进行拼接成一个字符串；

可以使用扩展运算符代替拼接；

返回拼接后的新数组；

```
let arr1 = [1,2,3];
let arr2 = [4,5,6];
let newarr = arr1.concat(arr2)
console.log(newarr);

let newarr = [...arr1,...arr2];
```

#### 翻转：`reverse()`

直接操作数组，忽略返回值；

```
let arr = [1,2,3];
arr.reverse();
```

#### 排序：`sort()`

- 底层为归并+插入的稳定排序，并且默认不是按数字排序，而是按字符串字典排序（会出现[1, 30, 4, 21, 100000].sort()，为[1, 100000, 21, 30, 4]的情况，排序数字时必须传入比较函数）


- 当比较函数return＞0为降序，return<0为升序（默认），return 0为不交换；
- 直接操作数组，忽略返回值；

```
let arr = [3,2,1];
arr.sort();//[1,2,3]

arr.sort((a, b) => b - a);//降序
```

#### 映射：`map()`

`array.map((element, index, array) =>  newElement);`

- 第一个参数：数组的每一个元素；
- 第二个参数：数组的每一个元素对应的索引；
- 第三个参数：原数组本身；

遍历数组处理数据，返回新数组；

```
const arr = [1, 2, 3];
const ret = arr.map(x => x * 2);

console.log(ret); // [2, 4, 6]
```

#### 遍历：`forEach()`

`array.forEach((element, index, array) => {// 副作用});`

单纯遍历数组访问数据，只做副作用；

- element只是arrayy的元素形参，array[index]才是实参；

返回值恒为 `undefined`

```
const arr = ['red','blue','green'];
arr.forEach(function(ele,index){
    console.log(ele);
})
```

#### 按元素查找：`indexOf()`

`array.indexOf(searchElement, fromIndex);`

在数组中查找某个元素第一次出现的位置（索引），找不到则返回-1；

不能用来查找NaN，不能用来查找对象（比较的是引用）；

可以用来判断元素是否存在（相当于`arr.includes(x);`）；

lastIndexOf：从后往前找；

- 第一个参数：要查找的元素值；
- 第二个参数：查找索引的起始位置，可以为负值（表示从 `length + fromIndex` 开始）；

```
const arr = ['red','blue','green'];
arr.indexOf('red');//0
```

#### 检测查找：`includes()`

`array.includes(searchElement, fromIndex);`

判断数组是否含有某个元素，返回布尔值；

严格匹配===，可以判断NaN是否存在；

- 第一个参数：要查找的元素值；
- 第二个参数：查找索引的起始位置，可以为负值（表示从 `length + fromIndex` 开始）；

```
const arr =[1,2,3,4];
arr.includes(2);//true;
arr.includes(2,1);//从索引为1开始判断是否有2
```

#### 条件查找：`find()`

`array.find((element, index, array) => {return boolean};`

在数组中查找第一个满足条件的元素本身，没有则返回undefined；

对于对象只要有一个属性符合，则返回整个对象；

回调返回 `true` 即命中，则立刻停止遍历，返回元素本身；

```
let arr = [1,2,3,4,5];
let result = arr.find(function(ele){
    return ele>2;//result=3
})
```

#### 筛选：`filter()`

`array.filter((element, index, array) => {return boolean;});`

返回一个符合指定条件的新数组，没有符合条件则返回空数组[ ]；

回调返回 `true` 保留元素，返回 `false` 则丢弃元素；

可以用来根据条件删除元素；

```
let arr =[1,2,3,4,5,6,7];
let newarr = arr.filter(function(ele){//数组存大于3的元素
     return ele>3;
})
```

#### 检测：`every() some()`

**every()：**检测数组所有元素是否都符合条件，符合则返回true，只要有一个不符合则false

```
let arr =[1,2,3,4,5,6,7];
let result = arr.every(ele => ele>0);//结果为true
```

**some()：**检测数组所有元素是否有一个符合条件，只要有一个符合则true

```
let arr =[1,2,3,4,5,6,7];
let result = arr.every(ele => ele>=7);//结果为true
```

#### 转化为字符串：`join() `

将数组的转化为字符串，也可以使用toString方法（底层就是jion()方法，只是不能设置分隔符）；

默认分隔符是逗号

```
const arr = ['red','blue','green'];
console.log(arr.join());//‘red,blue,green'，
join('');//没有分隔符
join('|');//分隔符为|
```

#### 累计处理：`reduce()`

`array.reduce(callback(accumulator, currentValue, currentIndex, array), initialValue)`

- 第一个参数：处理函数（参数分别是当前累计的值，当前数组元素，当前数组元素索引，数组本身）；
- 第二个参数：累计初始值

返回累计处理的结果

```
let arr =[1,2,3,4];
let a =arr.reduce(function(prev,current){
    reutrn prev + current;//返回累加结果，n=10
})

let b = arr.reduce(function(prev,current){
    reutrn prev + current;
},10)//有初始值b = 20
```

#### 扁平化：`flat()`

将高维数组转化为一维数组，flat可携带参数，表示扁平化到第几层

```
let a = [[12, 12], [24, 24,256],[35]];
let b = a.flat();//b为[ 12, 12, 24, 24, 256, 35 ]
```

#### 伪转真`Array.from()`

将伪数组或可迭代对象转换为真正的数组；第一个参数为伪数组对象或可迭代对象，第二个参数为映射函数（用于在转换为数组的同时，对每个元素进行加工处理），第三个参数为映射函数的this 指向；

伪数组：类似数组，但本质是含有length属性的普通对象，键名是 0、1、2... 这样的数字（或字符串形式的数字，如 `'0'`、`'1'`），所以一个只有length属性的对象可以认作为伪数组；

```
let lis = document.querySelectorAll('li');
let newlis = Array.from(lis);

const arr = Array.from({ length: 100 }, (_, i) => i);
```

#### 判断`Array.isArray()`

用于判断该值是否为数组，伪数组为false

```
Array.isArray(value)
```

# String字符串对象

字符串一旦创建，其内容是不能被修改的，当修改字符串时，js引擎实际上会在内存中创建一个新的字符串，原来的字符串还在内存当中（通过str[0]只能访问，不能修改）；相同值得字符串会指向同一内存；

```
let str = new String('你的字符串');

let str = '你的字符串';

let str = "hello";
str[0] = "H";
console.log(str);//hello
```

#### 长度：`length`

```
str.length;//字符串的字符数
```

#### 查找：`indexOf`

根据指定字符找对应索引号

```
str.indexOf('你要查找的字符');//找不到返回-1，可用此来判断字符串是否有字符
str.indexOf('你要查找的字符',起始位置);
str.lastIndexOf//从后往前查找
```

#### 匹配1：`startsWith()`

匹配字符串是否以某个子串开头，是则返回true，否则返回false

```
let str = 'hello word';
str.startsWith('hello');//结果为true
str.startsWith('word',6);//结果为true，表示从第6个字符开始匹配
```

#### 匹配2：`includes()`

匹配字符串是否包含特定的子串，是返回true（区分大小写）

```
let str = 'hello word';
str.includes('hello');//结果为true
```

#### 截取：`substring()`

根据索引号截取字符串

```
let newstr = str.substring(startindex，endindex);
let newstr = str.substring(2);//从索引为2开始截取
```

#### 替换：`replace`

只能替换第一个符合条件的；

```
let str = "hello";
let newstr = str.replace("h","H");
console.log(newstr);
```

#### 字符串转为字符数组：`split`

```
let str = 'red,blue,green';
let newstr = str.split('分隔符');//分隔符根据str来设置
let newstr = str.split('');//直接转化为单个字符的数组
```

#### 重复：repeat



```
const str = 'hello world';
console.log(str.repeat(2));//hello worldhello world
```

#### 剪切空格：`trim()`

*从字符串的两端移除空白字符，并返回一个新的字符串;*要返回一个仅从一端修剪空白字符的新字符串，请使用 `trimStart()`或 `trimEnd()`

```
const greeting = "   Hello world!   ";
console.log(greeting.trim());//"Hello world!";
```





# Map键值对对象：

一种键值对（key–value）数据结构，用于存储映射关系，是哈希表的抽象；

一个键只能出现一次，同一键名的值会被覆盖，键可以是任何数据类型；

收集元素：键由相同性质的元素生成，值使用数组来存储相同性质的元素；

```
function fn() {}

let map = new Map([
    ['a', "one"],
    [123, "two"],
    [true, "three"],
    [fn,'four'],//键为函数
]);
```

**`set(key, value)`** 

根据键在末尾添加 / 修改值，支持链式调用；

```
map.set('a', 1);
map.set('a', 2); // 覆盖

map.set('a', 1).set('b', 2);
```

 **`get(key)`**

读取

```
map.get('a'); // 1
map.get('notExist'); // undefined
```

 **`has(key)`** 

根据键判断是否存在，返回布尔值；

可用此方法缓存数据，判断某个数据是否已经被操作过了

```
map.has('a'); // true
```

**`delete(key)`**

根据键删除

```
map.delete('a'); // true / false
```

**`clear()`** 

 清空

```
map.clear();
```

**`size`** 

元素数量

```
map.size; 
```

#### 遍历Map

**for of **

```
for (const [key, value] of map) {
  console.log(key, value);
}
```

**forEach**

```
map.forEach((value, key) => {
    console.log(value,key)
})
```

**keys()` / `values()` / `entries()**

```
map.entries()//获得键值对的map迭代器对象
map.keys()//获得键
map.values()//获得值

[...map.keys()];
[...map.values()];
[...map.entries()];
```

#### WeakMap

特殊的键值对集合，键必须是对象，没有`keys()`,`value()`,`entries`()方法，没有size属性；键是弱引用，不影响垃圾回收

```
let weakmap = new WeakMap();

//只能用set方法创建
weakmap.set({ a: 'aaa' }, 'aaa');
```
