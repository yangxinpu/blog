# Built-in Objects

Base class for all objects; Object's keys can only be string or symbol type; if written as other types, automatically converts to string;

Access object properties using dot (`.`) or bracket (`[]`) notation:

**Dot (`.`) access:**

- Property name must be a valid identifier, can't start with number, have spaces, special characters, etc.
- Cannot access dynamically: When accessing, JS directly looks for property name corresponding to key string

```js
let key = 'name';
console.log(obj.key); // undefined
```

**Bracket (`[]`) access:**

- Property name can be any string or expression
- Can access dynamically: `[]` can contain variables, expressions, strings, etc.; JS first calculates key's value, then looks for that property in object

```js
let key = 'name';
console.log(obj[key]); // ✅ "Alice"
```

#### delete

An operator that can delete properties based on object's key (actually converts to undefined)

```js
// Delete object property
const obj = { a: 1, b: 2 };
delete obj.a;
console.log(obj.a);

// Delete array element
const arr = [10, 20, 30];
delete arr[1];
console.log(arr); // [10, undefined, 30] (element deleted, but array length unchanged)
```

#### hasOwnProperty()

Determines if property is on object itself, not on prototype chain, based on property name; returns bool value; can only check first-level properties;

```js
let obj = {
  name: 'aaa',
  age: 18,
  hobby: ['basketball', 'pingpong'],
  family: {
    father: 'bbb',
    mother: 'ccc',
  },
};
console.log(obj.hasOwnProperty('name')); // true
console.log(obj.hasOwnProperty('father')); // false
```

#### Object.getOwnPropertySymbols()

Can get array of Symbol keys in object;

```js
const s1 = Symbol('id');
const s2 = Symbol('type');

let obj = {
  [s1]: '1',
  [s2]: 'bbb',
  name: 'aaa',
};

let symbolArr = Object.getOwnPropertySymbols(obj); // Outputs [Symbol(id), Symbol(type)]
console.log(symbolArr[0]); // Outputs Symbol(id)
console.log(obj[s1]); // Outputs 1
```

#### Object.keys()

Gets array of all property name strings in object;

```js
let obj = { name: 'aaa', age: 18 };
let arr = Object.keys(obj);
console.log(arr); // arr is "name" "age"
```

#### Object.values()

Gets all property values in object, returns an array;

```js
let obj = { name: 'aaa', age: 18 };
let arr = Object.values(obj);
```

#### Object.entries()

Gets 2D array of object property keys and values

```js
let obj = {
  a: 1,
  b: 2,
  c: 3,
};
console.log(Object.entries(obj)); // Outputs [['a', 1], ['b', 2], ['c', 3]]
```

#### Object.assign()

Shallow copies properties from one object to another; same properties will be overwritten;

```js
let obj1 = { name: 'aaa', age: 18 };
let obj2 = {};
Object.assign(obj2, obj1); // Add obj1's name, age to obj2
```

#### Object.defineProperty()

Defines new property for object, or modifies existing property, and can set various property characteristics (like whether modifiable, enumerable, etc.); first parameter is object to operate on, second is property name to operate on, third is property descriptor configuration (including data descriptor and accessor descriptor);

**Data Descriptor:**

writable: Whether property value can be modified, default false;

enumerable: Whether can be enumerated (like by `for...in`, `Object.keys()`), default false;

configurable: Whether property can be deleted or descriptor modified, default false;

```js
let obj = {
  a: 1,
  b: 2,
  c: 3,
};
Object.defineProperty(obj, 'd', {
  value: 4,
  enumerable: false,
  configurable: true,
  writable: true,
});
```

**Accessor Descriptor**

get: Function automatically triggered when accessing a property, used to return property value;

set: Function automatically triggered when modifying a property, used to handle property value update;

Cannot specify both value/writable and get/set in Object.defineProperty;

```js
let obj = {
  a: 1,
  b: 2,
  c: 3,
};
Object.defineProperty(obj, 'd', {
  enumerable: true,
  configurable: true,
  get: function () {
    return 4;
  },
  set: function (value) {
    console.log(value);
    return value;
  },
});
console.log(obj); // Outputs { a: 1, b: 2, c: 3, d: [Getter/Setter] }
```

#### Object.getOwnPropertyDescriptor()

Gets property descriptor for a property of an object

```js
let obj = {
  a: 1,
  b: 2,
  c: 3,
};
console.log(Object.getOwnPropertyDescriptor(obj, 'a'));
// Outputs { value: 1, writable: true, enumerable: true, configurable: true }
```

#### Object.freeze()

Makes object completely immutable (including adding, deleting, modifying properties); can use Object.isFrozen to check if frozen;

```js
let obj = {
  a: 1,
  b: 2,
  c: 3,
};
Object.freeze(obj);
console.log(Object.isFrozen(obj));
```

#### Object.seal()

Restricts object structure modification (like deletion and addition), but allows modifying existing property values; can use Object.isSealed to check if sealed;

```js
let obj = {
  a: 1,
  b: 2,
};
Object.seal(obj);
console.log(Object.isSealed(obj));
```

# Number Object

#### toFixed()

Formats number to specified number of **decimal places as string**, default parameter is 0, result rounds

```js
let float = 3.1415926;
let result = float.toFixed(2); // Keep two decimal places, result is 3.14
```

#### toString()

Converts number to **string** in specified base, default decimal, base range 2 to 36; originally method on Object.prototype, but Number overrides it

```js
let a = 10;
console.log(a.toString(2)); // Binary, result is 1010
```

#### Number.isInteger()

Static method, determines if passed value is integer; if value is not a number or is NaN, Infinity, also returns false;

```js
let a = 10;
console.log(Number.isInteger(a)); // true
```

# Array Object

Can declare array via new Array, parameter can be array length or array elements

```js
let arr = new Array(5);
let arr = new Array(5).fill(0); // Length 5, filled with 0

let arr = new Array(1, 2, 3, 4); // [1, 2, 3, 4]

let arr = new Array(); // []
```

#### Push: `push()`

Returns new length value;

```js
let arr = [];
let a = {
  name: 'aaa',
  age: 13,
};
let b = {
  name: 'bbbb',
};
// Append complex data types
arr.push(a);
arr.push(b);
arr.push({
  name: 'cccc',
  age: 18,
});
```

#### Unshift: `unshift()`

Returns new length value;

```js
let arr = [1, 2, 3, 4];
arr.unshift(-1, 0); // arr becomes [-1, 0, 1, 2, 3, 4]
```

#### Pop: `pop()`

Returns deleted element's value;

```js
let arr = [1, 2, 3, 4, 5];
arr.pop(); // arr becomes [1, 2, 3, 4]
```

#### Shift: `shift()`

Returns deleted element's value;

```js
let arr = [1, 2, 3, 4, 5];
arr.shift(); // arr becomes [2, 3, 4, 5]
```

#### Splice: `splice()`

`array.splice(start, deleteCount, item1, item2, ...)`

- First parameter: Operation's starting index position;
- Second parameter: Number of elements to delete from starting position;
- Following parameters: Elements to insert starting from before the starting position;

Operates on array directly, ignore return value;

```js
let arr = [1, 2, 3, 4, 5];

// Delete, returns array of deleted elements
arr.splice(0, 3); // arr becomes [4, 5]

// Insert, second parameter 0 means only insert 6, 7, 8 starting at index 2, without deleting elements
arr.splice(2, 0, 6, 7, 8); // arr becomes [1, 2, 6, 7, 8, 3, 4, 5]

// Replace, second parameter not 0 means delete 2 elements starting at index 2 then insert 6, 7
arr.splice(2, 2, 6, 7); // arr becomes [1, 2, 6, 7, 5]
```

#### Concat: `concat()`

Cannot directly use `+` as it converts both arrays to strings then concatenates into one string;

Can use spread operator instead of concatenation;

Returns new concatenated array;

```js
let arr1 = [1, 2, 3];
let arr2 = [4, 5, 6];
let newarr = arr1.concat(arr2);
console.log(newarr);

let newarr = [...arr1, ...arr2];
```

#### Reverse: `reverse()`

Operates on array directly, ignore return value;

```js
let arr = [1, 2, 3];
arr.reverse();
```

#### Sort: `sort()`

- Underlying is stable sort using merge + insertion; by default sorts by string dictionary order, not numerically (can result in [1, 30, 4, 21, 100000].sort() becoming [1, 100000, 21, 30, 4]; must pass comparison function when sorting numbers);
- When comparison function return > 0 is descending, return < 0 is ascending (default), return 0 is no swap;
- Operates on array directly, ignore return value;

```js
let arr = [3, 2, 1];
arr.sort(); // [1, 2, 3]

arr.sort((a, b) => b - a); // Descending
```

#### Map: `map()`

`array.map((element, index, array) => newElement);`

- First parameter: Each element of array;
- Second parameter: Index corresponding to each element of array;
- Third parameter: Original array itself;

Traverses array to process data, returns new array;

```js
const arr = [1, 2, 3];
const ret = arr.map((x) => x * 2);

console.log(ret); // [2, 4, 6]
```

#### ForEach: `forEach()`

`array.forEach((element, index, array) => { // side effects });`

Simply traverses array to access data, only does side effects;

- element is just array element's formal parameter; array[index] is the actual parameter;

Return value is always `undefined`;

```js
const arr = ['red', 'blue', 'green'];
arr.forEach(function (ele, index) {
  console.log(ele);
});
```

#### Find by Element: `indexOf()`

`array.indexOf(searchElement, fromIndex);`

Finds first occurrence position (index) of an element in array, returns -1 if not found;

Cannot use to find NaN, cannot use to find objects (compares references);

Can use to determine if element exists (equivalent to `arr.includes(x)`);

lastIndexOf: Searches from back to front;

- First parameter: Element value to find;
- Second parameter: Starting position for search, can be negative (means starting from `length + fromIndex`);

```js
const arr = ['red', 'blue', 'green'];
arr.indexOf('red'); // 0
```

#### Includes: `includes()`

`array.includes(searchElement, fromIndex);`

Determines if array contains certain element, returns boolean;

Strict match ===, can determine if NaN exists;

- First parameter: Element value to find;
- Second parameter: Starting position for search, can be negative (means starting from `length + fromIndex`);

```js
const arr = [1, 2, 3, 4];
arr.includes(2); // true
arr.includes(2, 1); // Start checking from index 1 if 2 exists
```

#### Find: `find()`

`array.find((element, index, array) => { return boolean });`

Finds first element itself in array that satisfies condition, returns undefined if none;

For objects, if any property matches, returns entire object;

Callback returns `true` to hit, immediately stops traversal, returns element itself;

```js
let arr = [1, 2, 3, 4, 5];
let result = arr.find(function (ele) {
  return ele > 2; // result = 3
});
```

#### Filter: `filter()`

`array.filter((element, index, array) => { return boolean; });`

Returns new array of elements meeting specified condition, returns empty array [] if none meet condition;

Callback returns `true` to keep element, returns `false` to discard element;

Can be used to delete elements based on condition;

```js
let arr = [1, 2, 3, 4, 5, 6, 7];
let newarr = arr.filter(function (ele) {
  // Array stores elements > 3
  return ele > 3;
});
```

#### Every / Some: `every()` `some()`

**every()**: Checks if all array elements meet condition, returns true if all meet, false if any doesn't

```js
let arr = [1, 2, 3, 4, 5, 6, 7];
let result = arr.every((ele) => ele > 0); // Result is true
```

**some()**: Checks if any array element meets condition, returns true if any meets

```js
let arr = [1, 2, 3, 4, 5, 6, 7];
let result = arr.some((ele) => ele >= 7); // Result is true
```

#### Convert to String: `join()`

Converts array to string; can also use toString method (underlying is join() method, just can't set separator);

Default separator is comma

```js
const arr = ['red', 'blue', 'green'];
console.log(arr.join()); // 'red,blue,green'
join(''); // No separator
join('|'); // Separator is |
```

#### Reduce: `reduce()`

`array.reduce(callback(accumulator, currentValue, currentIndex, array), initialValue)`

- First parameter: Handler function (parameters are current accumulated value, current array element, current array element index, array itself);
- Second parameter: Accumulator initial value

Returns accumulated processing result

```js
let arr = [1, 2, 3, 4];
let a = arr.reduce(function (prev, current) {
  return prev + current; // Returns accumulated result, n = 10
});

let b = arr.reduce(function (prev, current) {
  return prev + current;
}, 10); // Has initial value, b = 20
```

#### Flatten: `flat()`

Converts high-dimensional array to one-dimensional array; flat can take parameter indicating how many levels to flatten

```js
let a = [[12, 12], [24, 24, 256], [35]];
let b = a.flat(); // b is [12, 12, 24, 24, 256, 35]
```

#### Pseudo to Real: `Array.from()`

Converts pseudo-array or iterable object to real array; first parameter is pseudo-array object or iterable object, second parameter is mapping function (for processing each element while converting to array), third parameter is mapping function's this binding;

Pseudo-array: Array-like, but essentially a plain object with length property, keys are 0, 1, 2... numbers (or string form numbers like '0', '1'), so an object with only length property can be considered a pseudo-array;

```js
let lis = document.querySelectorAll('li');
let newlis = Array.from(lis);

const arr = Array.from({ length: 100 }, (_, i) => i);
```

#### Check: `Array.isArray()`

Used to determine if value is an array, pseudo-array returns false

```js
Array.isArray(value);
```

# String Object

Once string is created, its content cannot be modified; when modifying string, JS engine actually creates a new string in memory, original string still exists in memory (can only access via str[0], cannot modify); strings with same value point to same memory;

```js
let str = new String('your string');

let str = 'your string';

let str = 'hello';
str[0] = 'H';
console.log(str); // hello
```

#### Length: `length`

```js
str.length; // Number of characters in string
```

#### Find: `indexOf`

Find corresponding index by specified character

```js
str.indexOf('character to find'); // Returns -1 if not found, can use this to determine if string has character
str.indexOf('character to find', start); // Start position
str.lastIndexOf; // Search from back to front
```

#### Match 1: `startsWith()`

Matches if string starts with certain substring, returns true if yes, false otherwise

```js
let str = 'hello word';
str.startsWith('hello'); // Result is true
str.startsWith('word', 6); // Result is true, means match starting from 6th character
```

#### Match 2: `includes()`

Matches if string contains specific substring, returns true if yes (case-sensitive)

```js
let str = 'hello word';
str.includes('hello'); // Result is true
```

#### Substring: `substring()`

Extracts string by index

```js
let newstr = str.substring(startindex, endindex);
let newstr = str.substring(2); // Extract starting from index 2
```

#### Replace: `replace`

Can only replace first match;

```js
let str = 'hello';
let newstr = str.replace('h', 'H');
console.log(newstr);
```

#### String to Character Array: `split`

```js
let str = 'red,blue,green';
let newstr = str.split('separator'); // Separator set according to str
let newstr = str.split(''); // Convert directly to single character array
```

#### Repeat: `repeat`

```js
const str = 'hello world';
console.log(str.repeat(2)); // hello worldhello world
```

#### Trim Whitespace: `trim()`

Removes whitespace characters from both ends of string and returns a new string; to return a new string trimmed from only one end, use `trimStart()` or `trimEnd()`

```js
const greeting = '   Hello world!   ';
console.log(greeting.trim()); // "Hello world!";
```

# Map Object

A key-value data structure for storing mapping relationships, abstraction of hash table;

A key can only appear once; same key name's value will be overwritten; key can be any data type;

```js
function fn() {}

let map = new Map([
  ['a', 'one'],
  [123, 'two'],
  [true, 'three'],
  [fn, 'four'], // Key is function
]);
```

**`set(key, value)`**

Add/modify value by key, supports chaining;

```js
map.set('a', 1);
map.set('a', 2); // Overwrite

map.set('a', 1).set('b', 2);
```

**`get(key)`**

Read

```js
map.get('a'); // 1
map.get('notExist'); // undefined
```

**`has(key)`**

Determine if exists by key, returns boolean;

Can use this method to cache data, determine if certain data has been processed

```js
map.has('a'); // true
```

**`delete(key)`**

Delete by key

```js
map.delete('a'); // true / false
```

**`clear()`**

Clear

```js
map.clear();
```

**`size`**

Element count

```js
map.size;
```

#### Traversing Map

**for of**

```js
for (const [key, value] of map) {
  console.log(key, value);
}
```

**forEach**

```js
map.forEach((value, key) => {
  console.log(value, key);
});
```

**keys() / values() / entries()**

```js
map.entries() // Get key-value pair map iterator object
map.keys()    // Get keys
map.values()  // Get values

[...map.keys()];
[...map.values()];
[...map.entries()];
```

#### WeakMap

Special key-value collection; keys must be objects; no `keys()`, `values()`, `entries()` methods, no size property; keys are weak references, don't affect garbage collection

```js
let weakmap = new WeakMap();

// Can only create with set method
weakmap.set({ a: 'aaa' }, 'aaa');
```

# Set Object

Elements in set can only appear once (can be used for array deduplication); reappearing elements are ignored

```js
let set = new Set([1, 2, 3, 4]);
```

#### Set Operations

#### Common Set Methods

Set cannot directly use index to access elements;

```js
let set = new Set([1, 2, 3, 4]);

set.add(5); // Add at end

set.delete(2); // Delete element by value

set.has(2); // Check if set contains certain element

set.size; // Get set length

console.log(set);
```

#### Traversing Set

**for of method**

```js
let set = new Set([1, 2, 3, 4]);

for (let k of set) {
  console.log(k);
}
```

**forEach() method**

```js
let set = new Set([1, 2, 3, 4]);

set.forEach((value) => {
  console.log(value);
});
```

#### Set to Array

**Spread operator**

```js
// Array deduplication
let set = new Set([1, 2, 3, 4]);
let arr = [...set];
console.log(arr);
```

**Array.from() method**

```js
let set = new Set([1, 2, 3, 4]);
let arr = Array.from(set);
console.log(arr);
```
